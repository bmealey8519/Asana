# Asana Clone

A desktop task management application inspired by Asana, built with .NET MAUI and C#.
Completed Summer 2025

## Features

- Create and organize projects
- Manage tasks
- Due dates
- User-friendly interface
- Cross-platform using .NET MAUI

## Technologies

- C#
- .NET MAUI
- Visual Studio 2022

## Installation

1. Clone the repository
2. Open Asana2.sln
3. Select Asana2.Maui as Startup Item
4. Build and run
